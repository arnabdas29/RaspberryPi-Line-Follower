# RaspberryPi-Line-Follower

Things required:

1. RaspberryPi3
2. DC Motors (2 nos)
3. HW-95 or L293D motor driver (1 nos)
4. IR sensors (2 nos)
5. Chassis (1 nos)
6. Power supply / Battery / Power bank
